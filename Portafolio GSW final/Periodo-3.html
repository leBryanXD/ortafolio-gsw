1-Cloud Services

Mediante el uso adecuado de recursos online disponibles en la web, se realizo una investigación  sobre plataformas de servicios en la nube, profundizando en los principales servicios que estas ofrecen para el despliegue de servidores web.

2-Application Gateway

Esta guia trato sobre como configurar una puerta de enlace para un conjunto de servidores web, utilizando los servicios disponibles en la plataforma de Microsoft Azure.

REVISAR TRABAJO

3-Equilibrador de Carga

El objetivo de esta guia era realizar el proceso para configurar un Equilibrador de Carga para varios Conjuntos de Disponibilidad, utilizando los servicios disponibles en la plataforma de Microsoft Azure.

REVISAR TRABAJO

4-Lanzar Instancia AWS con Servidor Web LAMP

 En esta guia se realizo el levantamiento de un servidor web LAMP,  la cual es de las herramientas más utilizadas para el despliegue de aplicaciones y sitios web dinámicos, en esta actividad  utilce los recursos de la plataforma AWS para configurar una Nube Virtual Privada (VPC) en la que además configure servicios de redes indispensables para la configuración de un entorno web, así como también lance una instancia EC2 en la que se instalo de manera remota un servidor Apache y PHP.

REVISAR TRABAJO

5-Application Load Balancer

En esta guia se pusieron en práctica los conocimientos sobre configuración de VPC, Grupos de Seguridad, Instancias e instalación de servidores web, además se configuraron nuevas subredes para diferentes zonas de disponibilidad en donde se ubicaron las instancias EC2 a crear, luego se configuro un Grupo de Destinos en los cuales se mapearon las instancias y finalmente se creo un Application Load Balancer que permite redireccionar las peticiones de los clientes.
